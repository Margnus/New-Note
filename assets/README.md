# Assets Directory

This directory contains application assets.

## Subdirectories

- **images/**: Application images
- **icons/**: Application icons
- **fonts/**: Custom fonts

## Instructions

### Icons
Place custom icons in the `icons/` directory.
Supported formats: SVG, PNG

### Images
Place images in the `images/` directory.
Supported formats: PNG, JPG, GIF

### Fonts
Place custom fonts in the `fonts/` directory.
Supported formats: TTF, OTF
