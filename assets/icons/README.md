## Required Icons

### Launcher Icon (app_icon.png)
- **Size**: 1024x1024px
- **Format**: PNG with transparency
- **Purpose**: Main application icon for app stores and home screen

### Adaptive Icons (Android)
- **Background**: solid color or image
- **Foreground**: icon with transparency

### App Store Assets
- Screenshots (various sizes)
- Promotional images
- App preview videos
